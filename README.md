# PhonePe-Payment-Insights-Dashboard
Interactive Power BI dashboard analyzing PhonePe transaction data using Excel, Power Query, and DAX. Features KPI cards, transaction trends, payment status analysis, user insights, and interactive visualizations.

---

## 📌 Project Overview

This project is an interactive **PhonePe Payment Insights Dashboard** built using **Power BI**. It analyzes transaction data to provide business insights into payment performance, user behavior, transaction trends, and service utilization.

The dashboard enables stakeholders to monitor KPIs, identify transaction patterns, compare payment statuses, and make data-driven decisions through interactive visualizations.

The objective of this dashboard is to:

- Monitor PhonePe transaction performance

- Measure payment success rate
- Understand customer usage patterns
- Identify top users by transaction value
- Compare weekday and weekend transaction behavior
- Analyze service-wise transaction value

##  📊 Dashboard Features

### 1. KPI Cards

Displays:

- Total Transactions
- Total Transaction Value
- Unique Users
- Successful Payment Rate

---

### 2. Monthly Transaction Trend

Visualizes:

- Monthly Transaction Count
- Monthly Transaction Value

Helps identify seasonal trends and business growth.

---

### 3. Payment Status Filter

Interactive slicer to filter dashboard by:

- Successful
- Pending
- Failed

---

### 4. Service Transaction Value Analysis

Compares transaction value across different services such as:

- Loans
- Insurance
- Money Transfer
- Recharge

---

### 5. Age Segment Contribution

Donut chart showing contribution of different age groups to total transactions.

---

### 6. Top 5 Users

Displays users with the highest transaction values.

---

### 7. Weekday vs Weekend Analysis

Compares transaction distribution between:

- Weekdays
- Weekends

---

### 8. Interactive Tooltips

Custom tooltip pages provide additional details when hovering over visuals.

##  📌 Key Insights

- PhonePe processed over **288K transactions**.
- Total transaction value exceeded **₹3.3 Billion**.
- Payment success rate remains extremely high.
- Loan transactions contribute the highest transaction value.
- Weekday transactions are significantly higher than weekend transactions.
- A small group of users contributes a large share of transaction value.
- Younger age groups represent the largest user contribution.
