# PhonePe-Payment-Insights-Dashboard
Interactive Power BI dashboard analyzing PhonePe transaction data using Excel, Power Query, and DAX. Features KPI cards, transaction trends, payment status analysis, user insights, and interactive visualizations.

---

## 📌 Project Overview

This project is an interactive **PhonePe Payment Insights Dashboard** built using **Power BI**. It analyzes transaction data to provide business insights into payment performance, user behavior, transaction trends, and service utilization.

The dashboard enables stakeholders to monitor KPIs, identify transaction patterns, compare payment statuses, and make data-driven decisions through interactive visualizations.
